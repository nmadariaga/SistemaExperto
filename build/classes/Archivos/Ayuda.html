EDAD:

Esta variable representa la edad de la población a la cuál va dirigido el proyecto de negocio que se está evaluando. El rango qu está permitido es de 18 a 65 años.

SUELDO:

Se debe ingresar el sueldo aproximado que tiene la población del lugar dónde se establecerá el negocio.

FORMA DE PAGO:

Señala cúal será la forma de pago principal que tendrá el negocio.

SEXO:

En esta variable se selecciona el sexo de la población a la que estará orientado el producto o servicio.

Por ejemplo: Una tienda de ropa pude ser exclusiva para hombre o para mujeres.

COSTO DE PRODUCCION:

Aquí se ingresa el costo de producción del producto o servicio.

UTILIDADES:

Aquí se debe ingresar las utilidades que se quieren obtener por ventas en forma de porcentaje (%).

Por ejemplo: Si el costo de producción de un producto es $5000, y quiere obtener $2500 de utilidades, el porcentaje que debe ingresar es 50%.

TIPO DE NEGOCIO:

Se despliegan 2 opciones donde se elige que tipo de negocio se evaluara, las opciones son Producto o Servicio.

NEGOCIO:

Después de seleccionar una opcion en TIPO DE NEGOCIO aqui se mostraran sus opciones correspondientes:

Producto: Comida, Vestuario, Decoración, Articulos de Aseo, Bebidas Alcoholicas, Electrónica.

Servicio: Locomoción, Reparto, Telefonía, Televisión, Internet.
